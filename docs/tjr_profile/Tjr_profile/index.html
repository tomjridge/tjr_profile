Module Tjr_profile

Simple profiling.

include Tjr_profile__.Types
type 'waypoint profiler = {
mark : 'waypoint -> unit;
get_marks : unit -> ('waypoint * int) list;
print_summary : unit -> unit;
}

Profiler ops:

  • mark: imperative mark command
  • get_marks: a list of all marked waypoints and the corresponding times at which they were marked
  • print_summary: print a human-readable summary (see Pretty_print)
val make_int_profiler : now:(unit -> int) -> int profiler

The now argument is a timer. This should be eg Core.Time_stamp_counter.(fun () -> now () |> to_int63 |> Core.Int63.to_int |> fun (Some x) -> x)

val make_string_profiler : now:(unit -> int) -> string profiler
val dummy_profiler : 'a profiler

Use this as a placeholder (eg via a reference) which is subsequently replaced by the main executable.
